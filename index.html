Anarchy Tools

an opinionatedly, unopinionated set of tools for building Swift projects
install | docs | source

The Anarchy Tools toolset provides a straight forward way to build your Swift tools and libraries. It takes a non-nonsense approach to getting things done. Best of all, it's built right on top of the swift-llbuild foundation.

(package
  :name "simple-sample"
  :tasks { :build { :name "sample"
                    :tool "atllbuild"
                    :sources ["src/**.swift"]
                    :output-type "executable" }})

Install

  1. Download the latest release of atbuild.
  2. Place it on your $PATH where your shell can find it (eg. ~/bin)
  3. You should now be able to run atbuild.

Documentation

The documentation is a little sparse right now, but do not fear, it is coming! You can always run atbuild --help to see the list of current tasks that can be run from the working directory.
